use the following shell command to get a directory listing that has all the directory entry information fields, where

  • all the spaces between the fields are compressed into single spaces
  • file ctime values are in long-iso format (using --time-style=long-iso)
  • owner and group values are nor displayed
  • backup files are not included in the listing
#!/bin/bash
ls -logB --time-style=long-iso | cut -d ' ' -f 1,2,3,4,5,6,7-
