sum codez: use rsync to update website

Here we are using rsync to update HTML_DOCUMENT_ROOT from the Raspberry Pi's HTML_DOCUMENT_ROOT which currently lives at /var/www/html/ on the Raspberry Pi micro-SD card. 

rysnc --exclude '*~' --exclude='*.zip' --exclude='*.pl' --exclude '^#' \
      -avzub --del /var/www/html/ ./
