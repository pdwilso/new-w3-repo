Getting the tree(1) Command

The tree(1) command is commonly included in many present-day GNU/Linux distritbutions.

If tree(1) is not included in the distribution you are using, you should be able to install it using the commands sudo apt install tree on any Debian-based systems.

If you do not have rights to the sudo command, OR if you do not have a Debian-based system, you will need to contact a system administrator or consult the documentation for your system to install software packages.

Using the tree(1) Command

Once tree(1) is installed, use the following command to generate a static index.html file for the current directory: tree -hH ./ > index.html

Additionally, the -o parameter to the tree(1) command may be used to specify an output file to use. tree -hH . -o index.html
