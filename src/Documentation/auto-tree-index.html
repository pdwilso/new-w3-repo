auto generate index files using tree(1)


	  tree -hH ./ > index.html
      
	-x	Stay on current file system
	-f	Print the full path prefix of each file
	-d	List directories only
	  
	-D	Print the date of the last modification time, or if
           	-c is used, the last status change time for the
           	file instead.

	-F 	Append a '/' for directories, a '=' for socket files, a '*'
         	for executable files, a > for doors (Solaris) and a
         	'|' for FIFOs, as per ls -F
	

	-R	Recursively crawl down the directory tree; at each level
	  	execute tree again, adding '-o 00Tree.html' as a new	
		option each time.

	-o filename  send output to filename
	-Q quote the names of the files in double quotes
	-h print the size of the file in human-friendly units

	--timefmt format
	       Prints (implies -D) and formats the date according to
	       the format string, which uses the strftime(3)
	       syntax.

	--dirsfirst
