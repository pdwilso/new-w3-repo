2023-06-26 — Working with a Javascript Countdown Timer example from W3 Schools

requirements

We will abstract the code given in the example, creating a portable widget which can be inserted in any web page by inclusion of the source code file within a containing HTML document.

Furthermore, this goal includes a requirement that the count-down timer module widget so included be multi-instanciatable - that is: we must be able to create mutiple timers from the same code within the same document but without namespace collisions for CSS, HTML, or Javascript.
