There are a number of sites that contributed to our understanding of this software - the Fluid Simulator, and its associated dependencies on WebGL or other graphics rendering technologies.

Below is a list of tabs we had open at the end of our search, once we had the Simulation operational on the https://sumkittehz.codes/ site. Note that some of these may be duplicated in sections of this document, below.

  • Jamie Won's WebGL Fluid Simulation seems a bit unusual in the context of all the implementations we have looked at. Wong goes into detail with the maths and to some extent with the codes to implement the maths, here, but without using any of the builder sites that most of these examples use (e.g. codepen.io etc)
  • How to add this as a background to personal website #62 — this is an open issue (2023-06-17) on PavelDoGreat's GitHub Project for the code.
  • cloydlau/webgl-fluid/README.md This is WebGL Fluid which "ES Module support for WebGL-Fluid-Simulation .
  • PavelDoGreat/WebGL-Fluid-Simulation/blob/master/index.html — This is yet another link into the master version of the WebGL Fluid Simulation provided by @PavelDoGreat on GitHub.
